Usage: ztk <COMMAND> [OPTIONS]

Commands:
 help            | Prints usage instructions
 create <TITLE>  | Creates new zettel
 list            | Lists zettels
 edit <ID>       | Opens zettel in text editor
 search          | Opens search for zettels
 use <DIRECTORY> | Sets directory as zettels' location

Options:
 -u --use <DIRECTORY> | Use specified directory instead of default location
